📐 Image Size Standards

Use these dimensions when uploading new product/category images:

Type Dimensions Aspect Ratio File Size Format
Product 600 × 450px 4:3 50-80 KB JPEG (85% quality)
Category 800 × 400px 2:1 80-120 KB JPEG (85% quality)

🛠️ How to Optimize Images

Option 1: Online Tools (Easy, No Install)

🌐 TinyPNG / TinyJPG

tinypng.com - Best compression

Steps:

  • 1. Visit tinypng.com
  • 2. Upload image from assets/img/products/
  • 3. Download optimized version
  • 4. Resize to 600×450 (product) or 800×400 (category) using option 2
  • 5. Replace original file

📐 Resize.Image

online-convert.com - Resize & compress

Or resizeimage.net

Option 2: Desktop Tools (Professional)

🎨 ImageMagick (Free, Command Line)

imagemagick.org

# Product image (600x450)
convert input.jpg -resize 600x450 -quality 85 product.jpg

# Category image (800x400)
convert input.jpg -resize 800x400 -quality 85 category.jpg

🖼️ XnConvert (Free, GUI)

xnview.com - Easy batch processing

Supports resize, compress, and batch operations

⚙️ FFmpeg (Free, Command Line)

ffmpeg.org

ffmpeg -i input.jpg -vf scale=600:450 -q:v 8 product.jpg

Option 3: Code Solution (Advanced)

Use PHP with GD library or Node.js with Sharp:

# Node.js (Recommended for batch)
npm install sharp

const sharp = require('sharp');
sharp('input.jpg')
.resize(600, 450)
.jpeg({ quality: 85 })
.toFile('output.jpg');

🔄 Batch Optimize (All Images at Once)

Using XnConvert (Easiest):

  1. Open XnConvert
  2. Add all files from assets/img/products/ folder
  3. Click Output tab
  4. Set output folder (same location)
  5. Click Actions → Add Resize:
    • For products: 600×450
    • For categories: 800×400
  6. Add JPEG Quality → 85
  7. Click Convert

Using PowerShell + ImageMagick (Fast):

# Batch resize all products
Get-ChildItem "assets/img/products/product_*.jpg" | ForEach-Object {
  convert $_.FullName -resize 600x450 -quality 85 $_.FullName
}

# Batch resize all categories
Get-ChildItem "assets/img/products/category_*.jpg" | ForEach-Object {
  convert $_.FullName -resize 800x400 -quality 85 $_.FullName
}

📤 Upload New Images (Manual Process)

Steps:

  1. Prepare image
    • Size: 600×450 (product) or 800×400 (category)
    • Format: JPEG
    • Quality: 85%
    • File size: 50-80KB
  2. Upload to assets/img/products/
    • Name: product_[ID].jpg or category_[ID].jpg
  3. Update database
    UPDATE products SET thumbnail = '/demo/tienha-drinks/assets/img/products/product_1.jpg' WHERE id = 1;
  4. Clear browser cache (Ctrl+Shift+Del)
  5. Verify on website
